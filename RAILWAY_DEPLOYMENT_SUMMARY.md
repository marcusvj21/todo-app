## What You Need to Do

### Manual Steps (Railway Console):

1. **Create Railway Project:**
   - Go to https://railway.app
   - Deploy from GitHub: `marcusvj21/todo-app`
   - Set root directory: `backend`

2. **Add PostgreSQL:**
   - Add PostgreSQL database to project
   - Railway will auto-generate `DATABASE_URL`

3. **Set Environment Variables:**
   ```bash
   NODE_ENV=production
   PORT=3000
   CORS_ORIGIN=https://todo-app-frontend-marcus-projects-249ca51a.vercel.app
   ```

4. **Deploy & Test:**
   - Wait for deployment to complete
   - Test health endpoint: `https://your-app.up.railway.app/health`
   - Copy backend URL for frontend configuration

5. **Update Frontend:**
   - Add backend URL to Vercel environment variables
   - Redeploy frontend

**Full instructions:** See `RAILWAY_DEPLOYMENT_GUIDE.md`
